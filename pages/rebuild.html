Rebuild the DSi

DSiLumina can rebuild the DSi, It's replace the DSi's firmware., You can rebuild the DSi with the following steps(now use cli)

  1. Run the following command to rebuild the DSi. 
        node cli.js rebuild && node cli.js stamp && cd /sdcard/ && cp -r /home/%username%/.DSiLumina/DSi/ /sdcard/ && ./check.sh | tee -a /home/%username%/.DSiLumina/DSi/log.txt
  2. Open ./dsilumina and select "DSi -> LuminaDS(i)" to reboot the DSi and automatic change the Patched DSi to LuminaDS(i).
  3. If you want to rebuild the DSi with the original firmware, you can run the following command. 
          cd ../ && node cli.js fetch -e <nand/nand-e/nand-j/nand-j-e> && cd /sdcard/ && cp -r /home/%username%/.DSiLumina/DSi/ /sdcard/ && rm -rf /home/%username%/.DSiLumina/DSi/ && ./check.sh | tee -a /home/%username%/.DSiLumina/DSi/log.txt
